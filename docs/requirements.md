# Requirements

## プロジェクト概要

AI Code Review SaaS は、ユーザーが貼り付けたコードに対してAIによるレビューを行い、改善点と修正方針を返すSaaSです。

MVPでは、まず個人利用を前提に、コードレビューの基本体験を最小構成で実現します。

## 開発目的

- コードレビューの初期フィードバックを素早く得られるようにする
- レビュー結果をもとに、Codexへ渡せる修正プロンプトを生成する
- 将来的なOpenAI API連携、履歴保存、GitHub連携に拡張しやすい土台を作る

## MVPで実現する機能

- コードを貼り付ける
- AIレビューを行う
- レビュー結果を表示する
- Codexへ渡す修正プロンプトを生成する
- レビュー履歴を保存する

## MVPでは実装しない機能

- GitHub連携
- ログイン機能
- チーム管理
- 課金機能
- 複数リポジトリの自動レビュー
- Pull Requestへの自動コメント

これらは将来機能として段階的に検討します。

## 想定ユーザー

- 個人開発者
- 学習中のエンジニア
- 小規模プロジェクトでコード品質を確認したい開発者
- CodexなどのAIコーディング支援ツールを使ってコード改善を行いたいユーザー

## 解決したい課題

- コードを書いた直後に、簡単なレビューを受ける手段が少ない
- AIに修正を依頼するためのプロンプト作成に手間がかかる
- 過去のレビュー結果を振り返りにくい
- GitHub連携などの大きな仕組みを作る前に、レビュー体験そのものを検証したい

## 成功条件

- ユーザーがコードを入力し、レビュー結果を確認できる
- レビュー結果にスコア、概要、指摘事項、Codex向け修正プロンプトが含まれる
- レビュー履歴を保存し、後から参照できる
- 将来のOpenAI API連携、GitHub連携、認証機能に拡張しやすい構成になっている
